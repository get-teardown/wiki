============
Sandbox Mode
============

.. meta::
   :description: Exploring the creative freedom of Sandbox Mode. Unlimited resources, spawning items, and testing destruction.
   :keywords: teardown sandbox, unlimited resources, spawning items, creative mode

What is Sandbox Mode?
=====================

Sandbox mode allows you to revisit any unlocked map without the pressure of a heist timer or objectives.

Features
========

* **Unlock Everything:** All tools are fully upgraded.
* **Unlimited Ammo:** Never run out of rockets or bombs.
* **Spawn Menu:** Spawn vehicles, props, and hazards.

How to Access
=============

1. Select "Sandbox" from the main menu.
2. Choose a map you have unlocked in the campaign.

Popular Activities
==================

* Testing structural integrity of buildings.
* Creating massive explosions without consequence.
* Building complex contraptions using planks and cables.
