Tools and Upgrades

Starter Tools
=============

**Sledgehammer**
The basic tool for breaking wood, glass, and weak masonry.
* *Upgrades:* Increase damage to break harder materials faster.

**Spray Can**
Used for drawing paths and marking areas. No upgrades needed.

**Fire Extinguisher**
Essential for putting out fires that you (inevitably) start.

Unlockable Tools
================

As you progress, you unlock more powerful equipment:

* **Shotgun:** Good for creating holes in wood and plaster.
* **Blowtorch:** Melts metal hinges and locks.
* **Planks:** Create bridges and ramps.
* **Pipe Bombs:** Moderate explosive damage.
* **Rocket Launcher:** Heavy destruction.

Upgrading
=========

Tools can be upgraded at your home base computer using money earned from heists. Prioritize upgrading **Plank** amount and **Shotgun** ammo early on.
