Compatibility
=============

Teardown is natively designed for Windows 10 (64-bit) and is fully compatible with Windows 11.

Requirements Checklist
======================

* **64-bit OS:** Required. 32-bit versions are not supported.
* **DirectX:** Version 12 is recommended for optimal ray tracing performance.
* **Drivers:** Ensure GPU drivers are updated to the latest version (NVIDIA Game Ready or AMD Adrenalin).

Windows 11 specifics
====================

If you are on Windows 11 and experience stuttering:

1. Disable "Game Mode" in Windows settings if you notice input lag.
2. Check your power plan is set to "High Performance".

Troubleshooting Windows Issues
==============================

If the game fails to launch:

* Run as Administrator.
* Install Visual C++ Redistributables.
