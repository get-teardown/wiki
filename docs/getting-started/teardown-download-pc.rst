Troubleshooting Common Issues
=============================

* **Low FPS:** Ensure your GPU drivers are updated. Lower the render scale in the game settings.
* **Crash on Launch:** Verify the integrity of game files via Steam properties.
* **Mod Conflicts:** Disable active mods one by one to identify the cause of instability.
